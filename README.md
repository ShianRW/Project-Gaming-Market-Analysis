# Project-Gaming-Market-Analysis
Global Gaming Market Penetration & Value Perception Analysis (Steam, PlayStation, Xbox)

## Data Source Credit
This project uses data from the Kaggle dataset:
"Gaming Profiles 2025 (Steam, PlayStation, Xbox)"  
Source: https://www.kaggle.com/datasets/artyomkruglov/gaming-profiles-2025-steam-playstation-xbox  
All credit to the original dataset creator: Artyom Kruglov

### Population Data
This project uses population data from **Our World in Data (OWID)** to support 
the market penetration analysis. The dataset includes global and country-level 
population estimates up to 2023.

**Citation:**
Ritchie, H., Roser, M., et al. (2023). *Population.* Our World in Data.  
Available at: https://ourworldindata.org/population
